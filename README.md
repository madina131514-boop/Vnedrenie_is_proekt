##Как использовать репозиторий
1.Все документы проекта хранятся в соотвествующих папках.
2.Названия файлов должны отражать их содержание(например,`TZ_vnedrenia.docx`).
3.При изменении документов создавайте новую версию,не удаляя старые (для сохранении истории).
